Privacy Policy

Last updated: 2026-07-14

1. Overview

Custom Widget is a personal Discord application used to display League of Legends statistics. This policy explains what data may be processed by the application.

2. Data collected or processed

The application may process the following data:

  • Discord user ID, if Discord account linking is used.
  • Riot ID, such as gameName and tagLine.
  • League of Legends statistics from Riot API, such as rank, LP, summoner level, champion mastery points, and recent match activity.
  • Technical data required to operate the application, such as message IDs, widget state, timestamps, and error logs.

3. Data not collected

The application does not intentionally collect Discord passwords, Riot passwords, payment information, private Discord messages, or sensitive personal documents.

4. Purpose of processing

Data is used only to display League of Legends statistics in connection with the Discord application and to maintain the technical operation of the service.

5. Storage

Data may be stored locally by the operator, for example in a SQLite database used by the tracker. Data may also be processed by Discord and GitHub when their services are used.

6. Data sharing

The application does not sell personal data. Data may be shared only with service providers needed to operate the application, such as Discord, GitHub, or Riot Games APIs.

7. Data deletion

To request deletion of stored data related to your account, contact the operator through GitHub.

8. Security

Reasonable efforts are made to protect API keys and stored data. Riot API keys are not intentionally exposed in client-side code.

9. Riot Games disclaimer

This project is not endorsed by Riot Games and does not reflect the views or opinions of Riot Games or anyone officially involved in producing or managing Riot Games properties.
